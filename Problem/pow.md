
## [Pow(x, n)](https://leetcode.com/problems/powx-n/)

### Problem Statement:
Implement pow(x, n), which calculates x raised to the power n (i.e., xn).

Example 1:

```

Input: x = 2.00000, n = 10
Output: 1024.00000

```

 

---------------------------------------------------------------------------------------------------
### Solution:

```c


double myPow(double x, int n)
{
    double i;
    i=pow(x,n);
    return i;

}


```
